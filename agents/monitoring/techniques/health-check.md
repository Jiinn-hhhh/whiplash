# 건강 상태 점검

- **대상 행동**: "이상 감지 시 보고"

---

## 점검 절차

1. `memory/monitoring/targets.md`에서 점검 대상을 확인한다.
2. 항목별 현재 값을 수집하고 임계치와 비교한다.
3. **이상 항목이 있을 때만** Manager에게 보고한다. 정상이면 보고하지 않는다.

## 상태 등급

| 등급 | 기준 | 행동 |
|------|------|------|
| 정상 | 임계치 미만 | 기록 안 함 |
| 주의 | 임계치 80%+ | 추이 관찰, 다음 점검 때 재확인 |
| 경고 | 임계치 위반 | 즉시 Manager 보고 |
| 위험 | 복수 위반 / 서비스 중단 | 즉시 Manager 보고, 최우선 |

## 보고 형식

정식 보고서 양식 불필요. 이상 항목만 짧게:
```
[항목]: 현재 값 / 임계치 — 원인 추정 (있으면)
```

## 모니터링 대상 관리

`memory/monitoring/targets.md`에서 관리. 항목 추가/삭제는 Manager 승인.
