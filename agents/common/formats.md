# 문서 양식 템플릿

에이전트가 사용하는 최소 양식. 과도한 양식 강제는 하지 않는다.

---

## 1. Handoff (Discussion → Manager)

`memory/discussion/handoff.md` — 전략 합의 + 실행 변경.

필수 필드:
- `User approved: yes`
- `Why this change`
- `Scope impact`
- `Manager next action`

---

## 2. 팀원 메모 (세션 간 맥락 보존)

`memory/{role}/` — 태스크 완료 시 핵심만 기록.

포함 내용:
- 어떤 파일을 왜 고쳤는지
- 주의할 점
- 다음에 이어할 때 알아야 할 것

형식은 자유. 과하게 쓰지 않되, 다음 세션에서 맥락 복원이 되는 수준.
