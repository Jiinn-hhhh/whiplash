# 지식 기여

- **대상 행동**: "태스크 완료 시 핵심 메모 남기기"

---

## 1. 메모 작성 시점

- release handoff 완료 후
- 장애 해결 후
- 아키텍처 결정 후
- 기술 부채 해결 후
- 반복되는 패턴 발견 시

---

## 2. 메모 내용

`memory/developer/`에 자유 형식으로 기록한다.

포함할 것:
- 어떤 파일을 왜 고쳤는지
- 주의할 점, 알려진 제약
- 다음에 이어할 때 알아야 할 것
- 구조적으로 반복 가능한 교훈

---

## 3. 산출물 공유

1. `workspace/teams/developer/` 내부에서 산출물을 작성하고 정리한다.
2. 확정된 산출물만 공유 대상으로 선정한다.
3. 중간 결과, 실험 빌드, 미검증 코드는 `workspace/teams/developer/`에 남긴다.
