# Ralph Loop

- **대상 행동**: "승인 대기 없는 조사/정리/재조사 반복"

---

## 기본 사이클

`search/docs -> synthesize -> gap 찾기 -> 재조사 -> 비교 정리`

- user 확인을 기다리며 멈추지 않는다.
- blocker가 생기면 manager에게 알리고, 다른 자료원/가설/범위 축소로 계속 진행한다.
- 조사 결과가 바로 실행에 닿으면 manager가 재계획할 수 있게 근거와 영향 범위를 분명히 적는다.

## 종료

- manager가 현재 랄프 종료 정책을 만족했다고 판단할 때만 완료로 본다.
